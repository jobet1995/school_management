How It Works

The global loading spinner automatically appears when any AJAX request starts and disappears when all requests complete.

  • Uses jQuery's global AJAX event handlers (ajaxStart, ajaxStop, etc.)
  • Tracks multiple concurrent requests
  • Provides visual feedback during loading states
  • Accessible with proper ARIA attributes
  • Responsive design for all screen sizes
